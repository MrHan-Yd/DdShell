# FinalShell Parity Matrix（对标补漏）

> 目的：基于公开可验证信息，建立功能对标矩阵，指导 Shell App 补齐替代能力。

## 1. 信息来源与可信度
- 官方主页特性（可验证）：`https://www.finalshell.cn/`
- 官方论坛升级说明（含高级/专业版功能）：`https://www.hostbuf.com/t/1081.html`

说明：第三方“破解版/镜像站”信息不作为主依据，仅作交叉参考。

## 2. 对标能力矩阵

### 2.1 核心能力（基础）
- SSH 连接：已规划。
- SFTP 双栏：已规划。
- 服务器监控（CPU/内存/网络/进程）：已规划。
- 命令历史/自动补全/参数化：已规划（命令中心 + 输入框增强）。
- 云端同步：已规划（自部署同步）。

### 2.2 高级/专业能力（补漏项）
- 多网卡网络监控：**待补**（P1）。
- 打包传输（自动压缩/解压）：**待补**（P1）。
- 高级网络监控（进程监听端口/连接状态）：**待补**（P1）。
- 高级进程管理（更多字段、筛选、操作）：**待补**（P1）。
- 无限命令历史/路径历史与快速切换：**部分已补**（继续细化）。
- 快速输入路径（命令输入框内选目录文件）：**待补**（P1）。
- 历史命令左右键字段选择输入：**待补**（P1）。
- 快速输入命令名称：**待补**（P1）。
- 终端背景图片个性化：**待补**（P2）。

## 3. 新增需求条目（补漏落地）
- FR-20 多网卡监控：支持选择并同时展示多个网卡上/下行。
- FR-21 打包传输：目录/多文件传输前自动打包，远端自动解压（可选）。
- FR-22 网络高级视图：展示端口监听与连接状态。
- FR-23 进程高级视图：支持更多列、过滤、排序与定位。
- FR-24 快速输入路径：命令输入框弹出路径树，选择后回填。
- FR-25 历史字段选择：左右键在历史命令字段间选择回填。
- FR-26 命令名快速输入：按关键字快速选命令名并回填。

## 4. 分期建议
- MVP：保持当前已规划范围（不膨胀）。
- v1.1：FR-20/21/24/26。
- v1.2：FR-22/23/25。
- v2：终端背景高级个性化与更多增强项。

## 5. 开放策略声明
- 本矩阵中所有对标能力均以“可免费使用”为目标。
- 即使能力来源于对标产品高级/收费版本，本项目也不设置付费门槛。
- 仅允许通过版本节奏（里程碑）控制上线先后，不允许商业授权控制功能可用性。
