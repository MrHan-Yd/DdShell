# Command Center Spec

## 1. 目标
- 提供高质感（macOS 风格）命令操作页面。
- 支持历史命令查看、检索、选择并回填终端输入。
- 明确“仅回填不执行”，避免误操作。

## 2. 页面结构
- 左侧：历史命令列表区。
- 顶部：搜索栏 + 筛选器（全部/当前主机/当前会话）。
- 右侧：命令预览与说明区。
- 底部：操作栏（插入终端、复制命令、收藏）。

## 3. 布局尺寸
- 页面最小宽度：`960px`。
- 左列表宽度：`360px`（可拖拽调整，范围 `320~480px`）。
- 行高：`36px`（紧凑模式 `32px`）。
- 卡片圆角：`12px`，浮层圆角：`14px`。

## 4. 视觉风格（macOS 质感）
- 背景：分层半透明面板（暗色优先），细边线 `1px`。
- 阴影：轻量双层阴影，避免厚重投影。
- 动效：
  - hover：`120ms ease-out`
  - 列表选中切换：`140ms ease-out`
  - 面板开合：`180ms cubic-bezier(0.2,0.8,0.2,1)`

## 5. 列表项定义
- 必填字段：`command`、`timestamp`、`hostLabel`、`exitCode?`。
- 可选字段：`tags[]`、`cwd`、`durationMs`。
- 展示规则：
  - 第一行显示命令（超长省略，hover 显示完整）。
  - 第二行显示时间 + 主机 + 标签。
  - 失败命令（`exitCode != 0`）显示警示点。

## 6. 交互规则
- 单击：选中列表项并在右侧预览。
- 双击：插入到当前终端输入区（不执行）。
- 回车：对当前选中项执行“插入终端”。
- `Cmd/Ctrl + F`：聚焦搜索框。
- `↑/↓`：移动选中项。
- `Esc`：关闭页面或清空搜索（优先清空搜索）。

## 7. 搜索与筛选
- 支持命令全文模糊匹配。
- 支持按标签筛选。
- 支持按时间范围筛选（今天/7天/30天）。
- 默认排序：按时间倒序。

## 8. 插入终端行为
- 插入目标：当前激活终端 tab。
- 插入方式：覆盖当前输入缓存（可配置为追加）。
- 插入后焦点：自动回到终端输入。
- 安全约束：默认不自动发送回车。

## 9. 状态与异常
- 加载态：骨架屏 + 占位行。
- 空态：提示“暂无历史命令”，附引导说明。
- 错误态：展示错误原因 + 重试按钮。
- 断线态：保留本地历史展示，禁用“插入终端”。

## 10. 可访问性
- 所有操作可键盘完成。
- 列表选中具备清晰高对比焦点态。
- 图标按钮提供 tooltip 与 aria-label。

## 11. 埋点建议（可选）
- `command_center_opened`
- `command_search_used`
- `command_inserted_to_terminal`
- `command_favorited`

## 12. 验收标准
- 2000 条历史命令下滚动与检索无明显卡顿。
- 键盘上下选择与回车插入稳定可用。
- 插入后命令仅回填不执行。
- 深浅色主题下视觉层次与可读性达标。
