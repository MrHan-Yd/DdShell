# 未来功能规划（候选）

说明：以下按可行性与收益做分层，避免一次性膨胀。P1 为可落地且高价值，P2 为中期增强，P3 为探索项。

## P1（可行性高 / 成本低到中）
- 连接诊断向导（本地检查与建议）；落地：本地检测 DNS/端口/认证/握手并输出建议。
- 连接模板与继承（减少重复配置）；落地：模板继承字段并允许覆盖，存储在 SQLite。
- 会话恢复与布局记忆（稳定性与连续性）；落地：持久化标签/分屏快照，启动提示恢复。
- 本地操作时间线（仅元信息，便于回溯）；落地：记录连接/传输/回填事件与时间戳。
- 连接提醒规则（生产环境提示）；落地：主机级开关 + 提示文案。
- 远端时间偏差提示 + 会话时间对齐显示（时间一致性）；落地：连接后拉取远端时间戳，状态栏显示偏差。
- 主机环境指纹 + 环境变更提示（漂移感知）；落地：保存 OS/内核/架构指纹并对比提醒。
- OpenSSH 配置导入/导出（减少重复维护）；落地：解析 `~/.ssh/config` 映射字段并支持导出。
- SFTP 目录书签（常用路径快速跳转）；落地：每主机保存书签列表并快速跳转。
- 主机头像/图标 + 连接标签颜色主题（识别效率）；落地：主机级图标与颜色配置入口。
- 连接状态仪表（延迟/质量快速感知）；落地：采集延迟/失败率并在状态栏展示。
- 传输冲突解决策略（覆盖/跳过/重命名模板）；落地：默认策略 + 每次可选覆盖。

## P2（可行性中 / 需要一定工程投入）
- 传输队列限速与调度（稳定传输体验）；落地：队列限速、时间窗与优先级。
- 传输失败自动恢复策略（减少人工干预）；落地：可配置重试次数与退避。
- 传输任务暂停恢复（断点续传与队列恢复）；落地：队列支持暂停/恢复与断点续传。
- 传输带宽统计（趋势与瓶颈判断）；落地：统计上传/下载流量与均速并图表展示。
- 连接生命周期报告（连接稳定性指标）；落地：记录连接耗时、重连次数与失败原因。
- 连接质量历史曲线（趋势与风险判断）；落地：保存延迟/失败率序列并展示趋势图。
- 连接状态颜色策略（动态质量标识）；落地：设阈值自动更新连接颜色标识。
- 主机标签与智能分组（规则引擎与冲突处理）；落地：规则自动分组并支持手动覆盖。
- 会话工作区（标签/分屏/主机集合一键恢复）；落地：保存快照并一键恢复。
- 片段库导入/导出（跨设备复用）；落地：本地文件导入/导出，支持合并或覆盖。
- SFTP 快速预览（文本/图片轻量预览）；落地：轻量预览与临时缓存控制。
- 风险护栏（命令提示与白名单）；落地：本地规则提示、可配置确认与白名单。
- 命令影响预估（风险等级 + 影响对象）；落地：解析关键参数并提示影响对象。
- 操作成本估算（基于历史的耗时/流量区间）；落地：基于历史速率估算区间值。

## P3（探索项 / 需验证价值或成本高）
- 任务化工作流（Recipe，多步命令编排）；落地：Recipe/Step 模型，逐步回填。
- 本地故障快照（失败时上下文采集）；落地：保存失败日志、网络状态与配置摘要。
- 主机侧边快速操作卡（每台主机快捷动作）；落地：每主机配置动作，回填不执行。
- 文件传输模板（路径组合一键应用）；落地：保存本地/远端路径组合模板。
- 多账号密钥切换助手（多凭据快速切换）；落地：同主机多凭据配置快速选择。

## 合并与取舍说明（可行性考虑）
- “风险护栏”与“命令影响预估”建议统一规则引擎，复用解析能力，减少维护成本。
- “连接状态仪表 / 颜色策略 / 生命周期报告 / 质量曲线”可共享同一套采集数据，分阶段开放。
- “时间偏差提示 + 会话时间对齐”建议合并为一个功能项，UI 上分两级展示。
- “主机头像/图标 + 标签颜色主题”建议一并实现，避免 UI 配置项过散。
