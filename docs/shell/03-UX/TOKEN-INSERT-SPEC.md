# Token Insert Spec

## 1. 目标
- 支持用户在终端输出区选中任意内容，并插入到当前命令光标处。
- 提升命令拼接效率（如 `docker rmi <image_id>`、`kill -9 <pid>`）。

## 2. 核心行为
- 选中来源：终端可见输出文本。
- 触发方式：右键菜单“插入到命令行”或快捷键触发。
- 插入位置：当前输入光标处（非整行覆盖）。
- 执行策略：仅回填，不自动发送回车。

## 3. 文本清洗规则
- 去除 ANSI 控制字符。
- 可配置是否保留换行（默认不保留，替换为空格）。
- 去除首尾空白。
- 路径含空格时可配置自动加引号。

## 4. 交互细节
- 插入成功后保持终端焦点。
- 若当前无可用输入会话，提示“当前会话不可输入”。
- 对超长文本插入前给出确认提示（可关闭）。

## 5. 异常处理
- 断线状态：禁用插入动作并展示重连提示。
- 无选中内容：提示“请先选中内容”。
- 清洗后为空：提示“选中内容不可插入”。

## 6. 验收标准
- 任意选中文本可稳定插入到光标处。
- 多次连续插入不丢失光标位置。
- 插入行为不触发命令自动执行。
- 含特殊字符内容插入后终端输入状态正常。
