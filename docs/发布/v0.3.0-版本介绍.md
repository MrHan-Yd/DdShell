# DdShell v0.3.0 发布

DdShell v0.3.0 基于 `v0.2.9..HEAD` 的实际提交整理，是一次围绕界面主题系统的重点版本。应用在 Classic 和 Aurora 之外新增 14 套完整主题，可选主题总数达到 16 套；主题选择与深色、浅色、跟随系统模式相互独立，并完整覆盖设置、连接管理、终端和 SFTP 等真实业务页面，让用户可以在不牺牲工具密度与可读性的前提下选择更适合自己的视觉语言。

## 亮点

### 16 套完整界面主题

本次新增以下 14 套主题：

- 渊泉：深海热泉、冷青与熔橙微光
- 曜砂：曜石黑、铂砂灰与冷青反射
- 云岚：天空蓝、云白米色与柔灰阴影
- 蓝栅：蓝图纸、电青制图线与工程网格
- 霜原：深靛、冰蓝、冷青与银白高光
- 锻墨：石墨黑灰、焊弧青与安全黄刻线
- 墨纸：宣纸白、墨色文字、朱砂与矿物蓝
- 珊光：深海蓝绿、贝母白与珊瑚点睛
- 竹影：深墨绿、竹叶绿与苔藓露光
- 星尘：深空黑紫、雾紫与低亮粉橙星尘
- 橙汐：深海蓝、海面青与日落橘高光
- 霓湖：雨夜湖面、冷青、青紫与玫红碎光
- 月蚀：近黑夜空、暗红本影与月铜边缘光
- 青瓷：瓷白空间、青釉层级、墨青细边与釉金

连同原有的 Classic 和 Aurora，DdShell 现在共提供 16 套可选主题。

### 主题和颜色模式独立保存

- 设置页新增完整主题卡片、视觉预览和当前主题概览
- 界面主题与深色、浅色、跟随系统模式相互独立，可自由组合
- 保存成功后才会切换全局视觉状态，避免草稿设置提前影响应用
- 已选主题通过 `ui.theme` 持久化，重启主窗口或快速编辑窗口后继续生效
- 中英文界面都提供主题名称和设计说明

### 真实应用页面完整适配

- 每套新增主题都覆盖标题栏、侧边栏、状态栏、弹层和通用控件
- 设置页覆盖主题卡片、表单、分段控件、切换按钮和保存浮栏
- 连接管理覆盖主机列表、详情面板、连接状态和操作按钮
- 终端覆盖标签、分屏、搜索、选择操作、断线提示和终端 ANSI 配色
- SFTP 覆盖目录树、文件列表、路径工具、传输抽屉和创建目录状态
- 支持深浅两套独立视觉，不使用简单反相代替暗色设计

### 更有辨识度的主题材质

- 统一主题选择卡片的选中边框、勾选状态和预览结构
- 青瓷加入瓷面内高光、稀疏开片线和窑夜暗色模式
- 霓湖加入低对比斜线碎光纹理，增强雨幕与湖面方向感
- 蓝栅保留工程网格与制图线，墨纸保留纸张和印记质感
- 优化主题名称，橘子海、星云尘埃、夜雨霓湖分别使用「橙汐」「星尘」「霓湖」作为中文显示名

## 其他改进

- 修复连接管理详情名称过长时挤压“已连接”状态和收藏、编辑、删除按钮的问题
- 长连接名称现在会单行省略，并可通过悬停查看完整名称
- 修正珊光主题分段控件宽度和多套主题的组件细节
- 补齐星尘主题早期遗漏的真实应用状态和完整选择器覆盖
- 增加主题原型与应用集成的结构规范，降低新增主题时漏改页面或持久化入口的风险
- 发布版本号同步更新到 `0.3.0`

## 已知限制

- macOS 未配置 Apple Developer 签名时，首次打开可能需要在系统“隐私与安全性”中手动放行，或执行发布页说明中的 `xattr -cr /Applications/DdShell.app`
- Linux 暂时继续使用 GitHub Releases 下载页作为应用内更新回退路径
- Windows 非默认安装目录和 `shell.db` 数据保留仍建议在发布完成后分别使用 MSI、NSIS 做一次真实升级烟测
