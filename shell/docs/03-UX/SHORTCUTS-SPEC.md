# Shortcuts Spec

本文件定义全局与页面级快捷键，保证跨平台一致体验。

## 1. 设计原则
- 同一动作在三平台语义一致。
- macOS 使用 `Cmd`，Windows/Linux 使用 `Ctrl`。
- 默认快捷键可重绑定。
- 冲突时以“终端输入安全优先”。

## 2. 全局快捷键
- 新建终端：`Cmd/Ctrl + T`
- 关闭当前标签：`Cmd/Ctrl + W`
- 切换标签：`Cmd/Ctrl + Shift + [` / `]`
- 新建连接：`Cmd/Ctrl + N`
- 命令面板：`Cmd/Ctrl + K`
- 打开设置：`Cmd/Ctrl + ,`
- 全局搜索：`Cmd/Ctrl + P`

## 3. 终端页快捷键
- 清屏：`Cmd/Ctrl + L`
- 分屏（水平）：`Alt + Shift + -`
- 分屏（垂直）：`Alt + Shift + |`
- 复制：`Cmd/Ctrl + Shift + C`
- 粘贴：`Cmd/Ctrl + Shift + V`
- 取词填充：`Alt + Enter`（选中终端内容后）

## 4. 文件页快捷键
- 上传：`Cmd/Ctrl + U`
- 下载：`Cmd/Ctrl + D`
- 新建目录：`Cmd/Ctrl + Shift + N`
- 刷新：`F5`
- 重命名：`F2`
- 删除：`Delete`

## 5. 命令中心快捷键
- 聚焦搜索：`Cmd/Ctrl + F`
- 选择上/下：`↑` / `↓`
- 插入终端：`Enter`（仅回填不执行）
- 关闭面板：`Esc`

## 6. 冲突优先级
1. 系统保留快捷键
2. 终端安全相关快捷键
3. 页面级快捷键
4. 全局快捷键

## 7. 可重绑定规则
- 禁止绑定系统保留组合。
- 禁止把“执行类动作”绑定到单键。
- 重绑定后需即时生效并持久化。

## 8. 验收标准
- 三平台默认快捷键可用。
- 同动作跨平台行为一致。
- 自定义快捷键重启后保留。
