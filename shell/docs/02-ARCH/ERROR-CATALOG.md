# Error Catalog

本文件统一定义错误码、触发条件、用户提示与恢复动作。

## 1. 连接与认证

### `AUTH_FAILED`
- 触发条件：用户名/密码/密钥不正确。
- 用户提示：认证失败，请检查凭据或认证方式。
- 恢复动作：重新输入凭据；切换认证方式；重试连接。

### `HOST_UNREACHABLE`
- 触发条件：主机不可达、网络断开、DNS 解析失败。
- 用户提示：主机不可达，请检查网络或主机地址。
- 恢复动作：检查地址与端口；网络恢复后重试。

### `NETWORK_TIMEOUT`
- 触发条件：握手超时或命令超时。
- 用户提示：网络超时，请稍后重试。
- 恢复动作：自动重试（指数退避）；手动重试。

### `FINGERPRINT_MISMATCH`
- 触发条件：known_hosts 记录与当前主机指纹不一致。
- 用户提示：主机指纹发生变化，连接已阻断。
- 恢复动作：人工核验后手动信任更新。

## 2. SFTP 与传输

### `SFTP_PERMISSION_DENIED`
- 触发条件：远端目录或文件权限不足。
- 用户提示：权限不足，无法执行当前文件操作。
- 恢复动作：切换目录；提升权限；联系管理员。

### `ARCHIVE_TRANSFER_FAILED`
- 触发条件：打包/解包失败。
- 用户提示：打包传输失败，请检查路径、权限或磁盘空间。
- 恢复动作：改为普通传输；检查空间与权限后重试。

## 3. 监控与采样

### `METRICS_UNSUPPORTED`
- 触发条件：目标系统缺少采集命令或不支持对应指标。
- 用户提示：当前系统不支持该监控项，已降级显示。
- 恢复动作：仅展示可用指标；允许用户关闭该项。

### `METRICS_PARSE_FAILED`
- 触发条件：命令输出格式变化导致解析失败。
- 用户提示：监控数据解析失败，部分图表不可用。
- 恢复动作：回退到兼容解析；提示用户反馈环境信息。

### `DISK_INFO_UNAVAILABLE`
- 触发条件：磁盘信息采集失败。
- 用户提示：无法获取磁盘使用信息。
- 恢复动作：重试采集；切换到基础监控视图。

### `NETWORK_INTERFACE_UNAVAILABLE`
- 触发条件：指定网卡不存在或不可用。
- 用户提示：网卡不可用，请重新选择网络接口。
- 恢复动作：自动回退到默认网卡；刷新网卡列表。

## 4. 命令提示与历史

### `COMMAND_HISTORY_UNAVAILABLE`
- 触发条件：历史命令读取失败或损坏。
- 用户提示：历史命令暂不可用。
- 恢复动作：回退到系统命令提示；允许重建历史索引。

## 5. Workflow Recipe

说明：本节对应的 UI / 产品名称为"命令宏"，错误码前缀保持 `WORKFLOW_*` 不变。
标注：标记为 [MVP] 的错误码在首版必须实现，标记为 [future] 的在后续增量实现。

### `WORKFLOW_RECIPE_NOT_FOUND` [MVP]
- 触发条件：流程模板不存在或已删除。
- 用户提示：流程不存在或已被删除。
- 恢复动作：刷新流程列表并重新选择。

### `WORKFLOW_RECIPE_INVALID` [MVP]
- 触发条件：流程模板静态校验失败，如空步骤、重复参数 key、非法 timeout。
- 用户提示：流程配置无效，请检查步骤和参数定义。
- 恢复动作：阻止保存并定位到错误字段。

### `WORKFLOW_RUN_NOT_FOUND` [MVP]
- 触发条件：查询运行记录时目标不存在。
- 用户提示：运行记录不存在。
- 恢复动作：刷新运行历史。

### `WORKFLOW_HOST_NOT_FOUND` [MVP]
- 触发条件：目标主机不存在或已被删除。
- 用户提示：目标主机不存在。
- 恢复动作：重新选择主机。

### `WORKFLOW_PARAM_MISSING` [MVP]
- 触发条件：启动流程时缺少必填参数。
- 用户提示：缺少必填参数，无法启动流程。
- 恢复动作：回到参数表单补全。

### `WORKFLOW_TEMPLATE_RENDER_FAILED` [MVP]
- 触发条件：模板语法非法，或引用了不存在变量。
- 用户提示：流程模板渲染失败，请检查变量定义。
- 恢复动作：修正模板或参数后重试。

### `WORKFLOW_EXEC_FAILED` [MVP]
- 触发条件：SSH exec 通道建立失败或执行失败且未命中更具体错误。
- 用户提示：流程执行失败，请查看步骤日志。
- 恢复动作：检查连接状态、参数与日志后重试。

### `WORKFLOW_UNSUPPORTED_INTERACTIVE_COMMAND` [MVP]
- 触发条件：检测到第一版不支持的交互式命令。
- 用户提示：当前流程包含交互式命令，暂不支持自动执行。
- 恢复动作：改写为非交互命令或手工执行。

### `WORKFLOW_RUN_ALREADY_FINISHED` [future]
- 触发条件：对已结束的运行记录继续执行取消等操作。
- 用户提示：当前运行已结束，无法继续该操作。
- 恢复动作：查看日志或重新运行。
- 说明：首版不实现取消，此错误码在取消功能落地后启用。

### `WORKFLOW_STEP_TIMEOUT` [future]
- 触发条件：步骤执行超过超时阈值。
- 用户提示：步骤执行超时。
- 恢复动作：查看日志，调整超时或重试。
- 说明：首版不实现超时控制，此错误码在超时功能落地后启用。

### `WORKFLOW_STEP_CANCELED` [future]
- 触发条件：用户主动取消运行。
- 用户提示：流程已取消。
- 恢复动作：允许重试整个流程。
- 说明：首版不实现取消，此错误码在取消功能落地后启用。

### `WORKFLOW_CONCURRENCY_LIMIT_REACHED` [future]
- 触发条件：同主机或全局并发超限。
- 用户提示：当前已有任务占用执行槽位，请稍后重试。
- 恢复动作：等待已有运行结束或取消后重试。
- 说明：首版同主机仅允许一个 running，超出直接返回 `WORKFLOW_EXEC_FAILED`，此错误码在并发队列功能落地后启用。

## 6. 通用处理规则
- 所有错误必须记录 `requestId` 便于追踪。
- 用户可恢复错误必须提供“重试”按钮。
- 阻断型错误必须提供明确风险说明（如指纹变化）。
- 错误文案避免技术黑话，优先给出可操作建议。
