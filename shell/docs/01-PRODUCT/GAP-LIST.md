# GAP LIST

用于记录当前文档体系与“全功能替代目标”之间的差距，并给出优先级。

## P0（建议近期补齐）
- 会话录制与回放（用于复盘与问题定位）
  - 价值：高
  - 复杂度：中
  - 建议：记录输入输出流并可回放。
- 端口转发 UI 细化（本地/远程/动态）
  - 价值：高
  - 复杂度：中
  - 建议：向导化配置与状态可视化。

## P1（可中期补齐）
- SSH 跳板链路可视化（多跳连接视图）
  - 价值：中
  - 复杂度：中
- 文件传输前冲突预览与策略模板
  - 价值：中
  - 复杂度：中

## Not Planned（明确不做）
- 插件机制/插件市场
  - 原因：增加架构复杂度与维护成本，不符合当前产品定位。
- 通用 IDE 式远端编辑器
  - 原因：项目定位仍然是 SSH / SFTP 专业工具，而不是项目级远端开发环境。
  - 说明：支持轻量 Quick Edit，但不做多文件工作区、LSP、Git 集成、项目树等 IDE 能力。
