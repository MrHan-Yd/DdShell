# 质量验收清单（参考）

说明：用于上线前自查与验收，聚焦可量化指标与高风险链路。

## 1. 性能
- 冷启动时间：首次启动 < 2.5s（Win/macOS/Linux）。
- 终端输入响应：键入到渲染 < 50ms（P95）。
- 候选提示延迟：输入变化到候选更新 < 100ms（P95）。
- 会话切换：标签切换 < 100ms（P95）。
- SFTP 列表加载：1000 文件 < 300ms（P95）。

## 2. 稳定性
- 断线重连：网络短断后自动重连成功率 > 95%。
- 任务恢复：传输中断后可恢复的任务比例 > 95%。
- 崩溃恢复：异常退出后可恢复会话/布局且无数据损坏。
- 长时运行：连续运行 24h 无明显内存泄漏或 UI 卡顿。

## 3. 体验一致性
- 输入法兼容：IME 组合输入不误触发、无卡死。
- 快捷键一致：跨平台主快捷键行为一致且可自定义。
- 焦点稳定：回填/插入后焦点回到终端输入区。
- 提示克制：风险提示不打断正常输入流，可配置关闭。

## 4. 安全与隐私
- 敏感信息：日志/崩溃报告默认脱敏或不写入敏感字段。
- 指纹提示：known_hosts 变更可感知且提示清晰。
- 本地数据：SQLite/配置文件不保存明文密码或私钥。

## 5. 可维护性
- 配置迁移：数据库/配置升级有版本号与回滚策略。
- 依赖管控：核心依赖锁定版本并有升级窗口策略。
- 错误分类：错误码/错误类型有统一规范，便于排查。

## 6. 测试覆盖（最低线）
- 终端：输入、复制粘贴、多行回填。
- 连接：成功/失败/重试/断线重连。
- SFTP：上传/下载/取消/失败恢复。
- 设置：偏好保存、重置、跨重启持久化。
- 片段：增删改、排序、匹配、回填。
