# 曜砂 UI 设计说明

## 名称

这套主题命名为「曜砂」，英文副名为 `Obsidian Sand`。

「曜」代表暗色界面里的微光、反射和冷静的金属感；「砂」代表细碎颗粒、磨砂表面和低饱和月白砂的材质感。它不是一个金色主题，而是一套偏深色、偏克制、带细微铂砂反光的工具型界面。

## 灵感来源

这套视觉从黑曜石、石墨灰、月白砂颗粒和冷青反射中提取色彩关系。核心不是把界面染成黄色，而是在黑色、中性灰和低饱和铂砂色之间建立层次，让重点操作、图标、选中态和焦点状态像被光扫到一样出现。

原始极光主题更强调紫青渐变和发光氛围。「曜砂」把这种强氛围收回来，转成更安静的金属材质语言：暗底、细边、低亮度扫光、局部高光。这样更适合 SSH、SFTP、监控、工作流这类高频工具界面，长时间使用时不会显得刺眼。

## 视觉原则

1. 深色为主，铂砂只做强调  
   主背景保持曜石黑和石墨黑，月白砂用于按钮、图标、选中态、焦点描边和少量装饰扫光。

2. 低饱和，不做黄滤镜  
   避免大面积黄色蒙层，使用月白砂、石墨灰、冷青反射和中性灰来建立质感。

3. 图标不重画，只统一材质  
   保留原有功能图标的形状语义，通过描边、暗底托盘、高光和 hover 状态统一成「曜砂」图标系统。

4. 工具优先  
   页面布局保持原型的密度和可扫读性，不把主题做成营销页或装饰页。

## 色彩结构

- 主背景：曜石黑、石墨黑
- 表面层：中性黑、深砂灰
- 主强调：月白砂、铂砂灰
- 辅助反射：冷青、金属灰绿
- 文本：暖白、灰米色、低亮度中性灰

## 适用场景

「曜砂」适合 DdShell 这类桌面开发工具：终端、连接管理、SFTP、监控、片段和工作流。它希望提供一种更精致但不浮夸的专业感，让主题有辨识度，同时不影响长时间操作。
