# 蓝栅 UI 设计说明

## 名称

这套主题命名为「蓝栅」，英文副名为 `Draftgrid`。

「蓝」代表工程蓝图纸、制图线和精确标注；「栅」代表网格、坐标、面板秩序和开发工具的结构感。它不是普通深蓝主题，也不是冰蓝、云蓝或霓虹蓝，而是一套以工程蓝图纸、电青制图线、冷白文字、蓝紫操作色和洋红校审线构成的工具型界面。

## 灵感来源

这套视觉来自建筑蓝图、机电管线图、CAD 坐标网格和图纸校审标记。大面积背景像深蓝晒图纸，界面边界像青色制图线，重要操作使用蓝紫高光，校审标记使用少量洋红，文字则使用冷白和蓝灰来保持清晰阅读。

DdShell 是高频开发工具，所以「蓝栅」不会把网格做成满屏装饰。网格、刻度、斜线和校注色只用于首页、logo、图标托盘、选中态和图表高光；终端、列表、文件表格和设置项仍以可扫读性为第一优先级。

## 视觉原则

1. 蓝图感来自结构，不靠一片蓝
   背景可以深，但需要有纸面层级、制图线和象牙白文字支撑，避免变成普通深蓝面板。

2. 青线负责秩序
   图标、边框、网格、信息态和图表辅助线使用青色，像工程图纸上的线稿。

3. 蓝紫负责操作，洋红只做校审
   主按钮、选中态和 focus 使用冷蓝紫，少量洋红只出现在校审线、图表区分和 logo 标记里，避免落回暖色海岸主题。

4. Logo 保留肥肥感
   保留双 D 的厚实轮廓，在内部加入坐标线、节点和蓝紫操作光，背景留足安全边距。

## 色彩结构

- 蓝图纸：深钴蓝、蓝黑、晒图蓝
- 制图线：电青、浅青、低亮网格蓝
- 操作色：蓝紫、靛蓝
- 校审色：洋红
- 测量提示：琥珀只用于 warning
- 纸面反光：冷白、蓝灰白
- 文本：冷白、冷灰蓝、低亮蓝灰

## 适用场景

「蓝栅」适合 DdShell 这类桌面开发工具：连接管理、终端、SFTP、监控、片段和工作流都可以共用。它补足现有主题里缺少的工程制图方向，既有鲜明识别点，又能保持专业、清晰和高密度操作体验。
