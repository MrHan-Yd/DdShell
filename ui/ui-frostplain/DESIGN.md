# 寒夜冰原 UI 设计说明

## 名称

这套主题命名为「寒夜冰原」，英文副名为 `Frostplain`。

`Frostplain` 指极夜下的开阔冰原：深靛蓝提供夜色和空间感，冰蓝、冷青与银白像薄冰边缘、低温反光和远处天光。它不是高冲击的能量主题，而是一套冷静、清爽、专业的代码工具主题。

## 灵感来源

灵感来自冬季极夜的冰面与低空冷光。大面积背景保持深靛蓝和蓝黑，面板像被雪光轻微照亮的金属冰层；交互色使用冰蓝和冷青，关键高光使用银白，避免把界面做成刺眼的纯蓝或纯白。

DdShell 是高频开发工具，所以「寒夜冰原」强调长时间可读性。它的气质偏冷静、克制，不追求火山或渊泉那种强冲击，而是让终端、文件列表、监控图表和工作流编辑器在清爽的低温色系里保持秩序感。

## 视觉原则

1. 深色要有空气感  
   主背景使用深靛蓝和蓝黑，而不是纯黑，让界面保留寒夜空间和层次。

2. 冰蓝只做焦点  
   冰蓝、冷青用于按钮、选中态、焦点描边、状态点和图表高亮，不大面积铺满。

3. 银白负责清爽  
   文本、Logo 高光和关键边线使用银白反光，让主题显得干净、专业，但不过度发亮。

4. Logo 像冰裂和雪光  
   保留双 D 识别，在内部加入极简冰裂线和低亮雪光，不使用复杂纹理。

## 色彩结构

- 夜色：深靛蓝、蓝黑、冷灰蓝
- 冰层：冰蓝、冷青、低饱和浅蓝
- 反光：银白、雪白、淡蓝白
- 表面层：深蓝灰、冷金属蓝
- 文本：银白、蓝灰、低亮冷灰

## 适用场景

「寒夜冰原」适合 DdShell 这类长时间使用的专业代码工具：终端、连接管理、SFTP、监控、片段和工作流都可以共用。它提供清爽、冷静、低干扰的视觉体验，适合需要稳定阅读和持续操作的开发环境。
