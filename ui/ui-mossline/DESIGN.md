# 竹影 UI 设计说明

## 名称

这套主题命名为「竹影」，英文副名为 `Mossline`。

「竹」提供清瘦、垂直、带节奏的线条秩序；「影」提供深墨绿底色和低亮度层次；`Mossline` 则强调苔藓、露光和细线材质。它不是普通护眼绿，也不是高饱和荧光绿，而是一套偏深色、偏自然、适合长时间使用的绿色工具主题。

## 灵感来源

灵感来自夜色里的竹林、石面苔痕和叶尖露光。大面积背景保持深墨绿和近黑绿色，界面表面像被竹影压暗的工作台；交互色使用竹叶绿和苔藓绿，关键高光用很少的冷白露光。

DdShell 是高频开发工具，所以「竹影」不把页面做成装饰性的自然风景。首页可以有竹节竖线、苔痕柔光和细网格；业务页面仍保持清晰、紧凑、可扫读，让绿色成为秩序感和放松感，而不是遮挡信息的滤镜。

## 视觉原则

1. 绿要深，不要亮屏护眼模式  
   主背景用墨绿和黑绿，避免大面积浅绿导致廉价或发灰。

2. 竹叶绿只做交互重点  
   按钮、选中态、焦点描边、图标和图表使用竹叶绿，面积保持克制。

3. 纹理来自竹节和叶脉  
   首页与 Logo 使用竖向竹节、细线叶脉和苔痕微光，不复用金属扫光或宇宙光带。

4. 工具感优先  
   保留终端、SFTP、监控和工作流页面的信息密度，图标按钮通过深绿托盘、浅绿描边和露光 hover 统一皮肤。

## 色彩结构

- 夜竹：近黑绿、深墨绿、低亮苔绿
- 表面层：黑绿、深叶绿、苔灰绿
- 主强调：竹叶绿、苔藓绿
- 高光：冷白露光、浅叶绿
- 文本：冷白、灰绿、低亮苔灰

## 适用场景

「竹影」适合 DdShell 这类长时间使用的桌面开发工具：连接管理、终端、SFTP、监控、片段和工作流都可以共用。它补足现有主题中缺少的绿色自然系，同时保持专业、克制和清晰。
