# 珊光礁 UI 设计说明

## 名称

这套主题命名为「珊光礁」，英文副名为 `Lumen Reef`。

「珊」来自珊瑚的暖色生命感，「光」来自浅水折射和微弱荧光，「礁」代表深海下稳定、安静、有层次的界面底座。它不是普通海蓝主题，而是一套深海蓝绿、贝母白、浅水青和珊瑚色共同构成的清透工具型界面。

## 灵感来源

这套视觉从夜色海礁、珊瑚枝、贝壳内壁和浮游微光中提取色彩关系。主界面以深海蓝绿建立稳定感，贝母白负责边缘高光和按钮反射，浅水青用于图标、信息态和光感，珊瑚色只放在关键操作、选中态、focus 和 logo 点睛处。

它需要比浅蓝主题更深、更有宝石感；比深色克制主题更清透、更有生命力；比暖色主题更冷静，不做大面积暖色铺陈。

## 视觉原则

1. 深海做底，珊瑚点睛  
   大面积背景保持低亮度蓝绿，珊瑚色只用于强操作和视觉锚点，避免界面变成粉橙主题。

2. 清透但不发灰  
   用贝母白和浅水青做提亮，减少雾化蒙层，保证长时间看仍然清晰。

3. 图标像被水光照到  
   图标保持原有语义，通过蓝绿托盘、贝母高光和珊瑚 hover 统一成「珊光礁」系统。

4. 工具优先  
   保持 SSH、SFTP、监控、工作流页面的密度和可扫读性。氛围只服务层次，不盖过信息。

## 色彩结构

- 主背景：深海蓝绿、暗礁青黑
- 表面层：礁石蓝、深水玻璃
- 主强调：珊瑚粉橙
- 辅助高光：贝母白、浅水青
- 文本：贝壳白、浅青灰、低亮度水雾灰

## 适用场景

「珊光礁」适合 DdShell 这类桌面开发工具：终端、连接管理、SFTP、监控、片段和工作流。它希望在专业工具界面里加入一点清透、冷艳、带生命力的视觉记忆点，同时不牺牲信息清晰度。
