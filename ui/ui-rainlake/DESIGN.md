# 夜雨霓湖 UI 设计说明

## 名称

这套主题命名为「夜雨霓湖」，英文副名为 `Rainlake`。

`Rainlake` 指雨夜里的深蓝黑湖面：远处城市霓虹被雨水和湖面打碎，留下低饱和的青紫、玫红和冷蓝反光。它和极光相邻，但不做天空光带，而是把视觉重心放在湿润、倒影、涟漪和城市夜色上。

## 灵感来源

灵感来自雨后的城市湖岸：天空和湖面接近深蓝黑，远处招牌和路灯在水面形成细碎反光；雨痕让高光变窄、变碎、变低亮。它不是明亮霓虹主题，而是把霓虹压到边缘、按钮、状态点、Logo 和图表高光里。

DdShell 是高频开发工具，所以「夜雨霓湖」保留专业工具的低干扰和清晰层级。背景更像湿润的深色玻璃，交互色像水面短暂闪过的青紫和玫红反光，漂亮但不喧闹。

## 视觉原则

1. 湖面要深  
   大面积使用深蓝黑和湿冷蓝灰，保证终端、列表和表格有稳定阅读对比。

2. 反光要碎  
   首页、Logo 和高光使用短线、雨痕、涟漪式反射，不做连续大面积霓虹带。

3. 玫红只做点睛  
   玫红用于少量焦点、图表和 Logo 反光，青紫作为主交互冷光。

4. 城市雨夜，不是舞台霓虹  
   色彩低饱和，光效贴着边界和水面走，避免界面显得浮夸或像娱乐场景。

## 色彩结构

- 夜色：深蓝黑、湖面蓝、冷灰蓝
- 雨面：湿黑、雾蓝、低亮蓝灰
- 霓光：青紫、冷青、少量玫红
- 反光：雨白、淡紫白、湿润高光
- 文本：冷白、蓝灰、低亮灰蓝

## 适用场景

「夜雨霓湖」适合 DdShell 这类长期使用的开发工具：终端、连接管理、SFTP、监控、片段和工作流都可以共用。它比极光更城市、更湿润、更低饱和，适合想要精致深色质感但不想牺牲阅读效率的界面方向。
