# 青瓷 UI 设计说明

## 名称

这套主题命名为「青瓷」，英文副名为 `Celadon`。

「青」来自低饱和的青釉、灰绿釉影和墨青边线；「瓷」代表温润、细腻、坚实而不过度发光的表面质感。它不是普通浅绿色主题，也不是护眼滤镜，而是一套以瓷白空间、青釉层级、开片细线和少量釉金构成的工具型界面。

## 灵感来源

这套视觉参考汝窑与龙泉青瓷的釉色关系：主体接近灰绿和天青，边缘因釉层厚度出现更深的墨青，局部开片形成细而不规则的自然纹路。少量温润釉金只用于状态和视觉点睛。

「青瓷」是浅色优先主题，但暗色模式不是简单反相，而是转为窑夜墨青、青黑釉面和低亮青釉高光。页面仍然保持开发工具所需的信息密度；瓷面感主要来自细边、内高光和克制阴影，不依赖模糊玻璃或大面积渐变。

## 视觉原则

1. 瓷白负责空间
   大面积背景使用低亮瓷白和淡灰绿，不用纯白铺满，长时间阅读保持柔和但清晰。

2. 青釉负责层级
   面板、按钮、图标托盘和选中态通过淡青釉、灰绿釉影和墨青边线区分，不做普通绿色蒙层。

3. 开片要少而自然
   首页和 Logo 使用稀疏、不规则的开片线，业务页面只保留釉面高光，避免纹理干扰信息。

4. 釉金只做点睛
   釉金用于 warning、图表区分和 Logo 一线高光，主交互仍由釉青与墨青承担。

5. 工具优先
   保留终端、SFTP、监控和工作流页面的紧凑密度，瓷面材质只服务层级和识别。

## 色彩结构

- 瓷面：瓷白、暖瓷白、淡灰绿
- 青釉：淡青釉、灰绿釉影、亮青釉
- 墨线：墨青、深青黑、青灰
- 点睛：釉金、低亮暖金
- 文本：深墨青、灰绿、青瓷白

## 适用场景

「青瓷」适合 DdShell 这类长时间使用的桌面开发工具：连接管理、终端、SFTP、监控、片段和工作流都可以共用。它补足现有主题中缺少的东方瓷釉材质方向，与天空浅色、纸张浅色和深绿自然系主题保持清晰边界。
