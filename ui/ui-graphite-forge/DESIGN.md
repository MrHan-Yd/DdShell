# 石墨锻 UI 设计说明

## 名称

这套主题命名为「石墨锻」，英文副名为 `Graphite Forge`。

「石墨」代表深灰、枪铁、磨砂金属和稳定的工具底色；「锻」代表焊弧、信号灯、刻痕、边缘高光和工业设备的受力感。它不是月蚀那种暗红戏剧感，也不是曜砂的砂砾质感，更不是工程图纸方向，而是一套偏机房、仪表、金属设备和运维控制台的硬核中性灰工业主题。

## 灵感来源

这套视觉来自服务器机柜、CNC 控制面板、黑色阳极氧化铝、磨砂石墨外壳、焊接电弧和安全警示标记。大面积界面使用清晰的石墨黑灰层级，图标和选中态使用焊弧青，数据辅助和图表区分使用信号蓝，安全黄只在 warning、进度高光和少量 logo 标记里出现。

DdShell 是高频开发工具，所以「石墨锻」不会把金属纹理做得很脏。它通过细密刻线、斜向安全纹、低饱和金属面板和克制的信号色建立气质；终端、列表、文件表格和设置项仍然以清晰、耐看、可扫读为第一优先级。

## 视觉原则

1. 石墨层级要清楚
   深色界面不能糊成一整片黑。底层、面板、卡片、浮层之间需要靠灰阶、边框和微弱高光区分。

2. 焊弧青负责操作
   主按钮、选中态、focus、图标悬停和状态栏重点信息使用焊弧青，像工业设备上的冷光指示。

3. 信号蓝负责数据
   图表、信息态、辅助线和第二层强调使用信号蓝，避免所有高光都堆在同一种青色上。

4. 安全黄只做警示
   黄色只用于 warning、少量测量刻度和 logo 安全纹，不参与大面积背景，避免变脏。

5. Logo 保留肥肥感
   保留双 D 的厚实轮廓，把内部网格替换成金属刻线、铆点和安全黄斜纹，背景留足安全边距。

## 色彩结构

- 石墨底：近黑、枪铁灰、磨砂深灰
- 操作光：焊弧青、冷绿青
- 数据光：信号蓝、低亮钢蓝
- 警示色：安全黄、危险红
- 金属反光：冷白、灰绿白
- 文本：冷白、灰绿、低亮灰

## 适用场景

「石墨锻」适合偏专业和高密度的运维、监控、终端、SFTP 和工作流页面。它补足现有主题里缺少的工业石墨方向，视觉上更硬、更克制，也更贴近服务器工具、机房控制台和工程设备。
