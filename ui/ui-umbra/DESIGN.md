# 月蚀 UI 设计说明

## 名称

这套主题命名为「月蚀」，英文副名为 `Umbra`。

`Umbra` 指月蚀中的本影区域：光被遮住，只剩近黑夜空、暗红阴影和月铜色边缘光。这个名字比直接叫 Eclipse 更克制，也更适合暗色工具主题。

## 灵感来源

灵感来自月全食时的低亮月面：天空接近纯黑，月亮主体被阴影吞没，只在边缘保留暗红、铜橙和少量灰白反光。它不是高饱和红色主题，而是把红铜色压到边缘、按钮、关键状态和图表高亮里。

DdShell 是高频开发工具，所以「月蚀」不做大面积红色铺底。背景保持近黑和深棕黑，文本用冷白和灰铜保证阅读，红铜光只作为危险感和高级感的视觉信号。

## 视觉原则

1. 黑要干净  
   主背景使用近黑夜空，不做满屏红雾，保证终端和列表的阅读效率。

2. 红只做边缘光  
   暗红和月铜色用于按钮、焦点、边框、图表关键线和 Logo 的月弧，不成为大面积背景色。

3. 危险感要克制  
   主题带一点压迫感，但不把所有状态都染红。成功、警告、错误仍保持清晰区分。

4. Logo 保持干净  
   保留双 D 识别，内部不加纹路和点缀，只通过暗红到月铜的渐变和低亮边缘光表达主题。

## 色彩结构

- 夜空：近黑、深棕黑、低亮灰黑
- 本影：暗红、深酒红
- 边缘光：月铜、暗橙铜
- 表面层：深棕黑、低亮红灰
- 文本：冷白、灰铜、低亮暖灰

## 适用场景

「月蚀」适合 DdShell 这类长时间使用的高级暗色方向：连接管理、终端、SFTP、监控、片段和工作流都可以共用。它在近黑底色里保留少量红铜边缘光，保持克制、危险感和工具界面的清晰度。
