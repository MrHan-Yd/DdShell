# 深海热泉 UI 设计说明

## 名称

这套主题命名为「深海热泉」，英文副名为 `Abyssal Vent`。

「深海」提供近黑、低亮、冷静的底色；「热泉」提供局部高能的青绿、硫黄和熔橙微光。它和原来的极光主题不同：极光是天空里的大面积流光，深海热泉是黑暗海底里的点状喷发和烟羽，视觉更沉、更硬核。

## 灵感来源

灵感来自海底热液喷口：周围是几乎无光的深海蓝黑，喷口附近会出现青绿色生物荧光、硫化物沉积的黄绿色，以及高温矿物带来的橙红边缘光。

DdShell 是开发工具，所以这套主题不把发光做成满屏装饰。大面积背景保持暗、稳、低噪声；发光只用于交互焦点、状态点、图表曲线、按钮和 Logo。这样既有自然景观的记忆点，又不会影响终端、列表和监控信息的可读性。

## 视觉原则

1. 暗处要够暗  
   背景以近黑深海蓝为主，避免整页灰亮，保留暗色主题该有的沉浸感。

2. 光只在该亮的地方亮  
   热泉青作为主交互色，硫黄绿和熔橙作为辅助能量色，只出现在按钮、图表、状态和首页喷口视觉里。

3. 质感来自喷口，不来自霓虹  
   光效不是均匀渐变，而是局部、高能、带方向的微光，像热液从海底喷出后向上扩散。

4. Logo 纹理来自热羽  
   保留双 D 识别，但左右两个 D 内部放入长短不一的极简上升热羽，中间保持分开，不再复用其它主题的内部纹路。

## 色彩结构

- 深海：近黑、深蓝黑、冷灰蓝
- 热泉：高亮青绿、硫黄绿、熔橙
- 表面层：低亮蓝黑、深青灰
- 强调色：热泉青
- 能量色：硫黄绿 + 熔橙
- 文本：冷白、灰青、低亮蓝灰

## 适用场景

「深海热泉」适合 DdShell 这类长时间使用的暗色开发工具：终端、连接管理、SFTP、监控、片段和工作流都可以共用。它比极光更克制、更低沉，但仍保留足够强的视觉识别点。
