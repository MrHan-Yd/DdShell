# 墨纸 UI 设计说明

## 名称

这套主题命名为「墨纸」，英文副名为 `Inkpaper`。

「墨」代表高对比、锋利文字和克制的黑色线条；「纸」代表温润的宣纸白、细微纤维和不刺眼的浅色底。它不是古风装饰主题，也不是黑白极简主题，而是一套带少量朱砂红和矿物蓝记忆点的浅色工具界面。

## 灵感来源

灵感来自宣纸、石墨、朱砂印和矿物蓝颜料。大面积背景使用低饱和纸白，文字和边界使用墨黑与石墨灰，主操作和选中态使用朱砂红，少量信息态和边缘光使用矿物蓝。

DdShell 是开发工具，所以「墨纸」不做书法、山水或厚重装饰。纸纹只能作为非常轻的背景质感；页面结构仍保持终端、列表、监控和工作流所需的清晰密度。朱砂红只用于焦点、按钮、Logo 印记和少量状态，不把界面染成红色。

## 视觉原则

1. 亮色为主，文字要锋利  
   主背景是纸白和米白，正文使用接近墨黑的颜色，保证长时间阅读不发灰。

2. 朱砂只做印记  
   主按钮、选中态、焦点描边和 Logo 记忆点使用朱砂红，面积小但识别强。

3. 矿物蓝负责冷静  
   信息态、边缘光和少量图表线使用矿物蓝，避免主题只有黑白红。

4. 纸纹要轻  
   首页可以出现细纤维、淡墨横压和印色痕迹，业务页面不做复杂纹理，保证工具界面干净。

## 色彩结构

- 纸面：宣纸白、米白、浅纸灰
- 墨色：墨黑、石墨灰、低亮暖灰
- 印记：朱砂红、浅朱红
- 辅助：矿物蓝、蓝灰
- 文本：墨黑、暖灰黑、低亮纸灰

## 适用场景

「墨纸」适合 DdShell 这类桌面开发工具的高质感浅色方向：连接管理、终端、SFTP、监控、片段和工作流都可以共用。它补足现有主题中缺少的纸白中性浅色风格，同时让图标按钮和操作重点有明确的朱砂记忆点。
