# 云岚 UI 设计说明

## 名称

这套主题命名为「云岚」，英文副名为 `Cloudrift`。

「天」代表高空、开阔、浅蓝与清透空气感；「云」代表柔边、层叠、米白阴影和轻盈漂浮感；「米」用于把云白压低一点，让界面不刺眼、不冷硬。主题重点不是普通米白，而是天光下的云层质感。

## 灵感来源

这套视觉参考浅色车身在天空漫反射下呈现的米白、雾蓝和柔灰层次。它不是高亮纯白，也不是暖黄米色，而是介于晴空浅蓝、云白和米色云影之间的柔和材质。

原始深色主题强调发光和金属感。「云岚」把重心改成天空和云：背景更轻、边界更软、阴影更薄、图标和按钮像浮在云层上的小型控制面。页面仍然保持开发工具所需的信息密度，但整体情绪更开阔、更安静。

## 视觉原则

1. 先有天，再有云  
   大面积背景以浅蓝灰和高空蓝建立空间感，米白只作为云层和面板表面出现。

2. 云要有层次  
   首页使用多层柔雾、云带和天光，不用单一米色铺底，避免变成普通浅色主题。

3. 轻，但不飘  
   组件边框、图标托盘和阴影都保持克制，让界面有漂浮感，但仍适合终端、SFTP、监控这类高频工具。

4. 图标延续统一皮肤  
   保留原图标语义，通过浅蓝、云米、柔高光和低对比托盘统一成「云岚」图标系统。

## 色彩结构

- 天：高空蓝、浅灰蓝、雾蓝
- 云：云白、米白、柔灰阴影
- 表面层：暖白、浅米、低饱和蓝灰
- 强调色：云米、天蓝高光
- 文本：深蓝灰、灰蓝、低亮度雾灰

## 适用场景

「云岚」适合 DdShell 这类桌面开发工具的浅色视觉方向：终端、连接管理、SFTP、监控、片段和工作流。它希望在专业工具里提供更轻、更清透的界面氛围，同时突出“天”的空间感和“云”的柔和层次。
