Tail latest nginx errors

实时跟踪 nginx error.log，仅过滤含 "error" 的行。常用于排查 502 / upstream timeout 问题。

nginx logs prod read-only
bash 
tail -f /var/log/nginx/error.log | grep -i "error"

Variables

$HOST api-prod-01 target host (auto-resolved from current connection)
$LOG_PATH /var/log/nginx/error.log log file to tail (override per host)
Read-only · safe to run on production
