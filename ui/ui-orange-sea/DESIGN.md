# 橘子海 UI 设计说明

## 名称

这套主题命名为「橘子海」，英文副名为 `Orange Tide`。

「橘子」不是单纯的黄色或橙色铺满界面，而是傍晚太阳压低以后落在海面上的暖光；「海」负责承载冷静、清晰和可读性。主题的核心不是高饱和橙色，而是日落橘、海面青、暖白反光与深海蓝之间的平衡。

## 灵感来源

灵感来自自然景象里的橘子海：太阳接近海平线，天空有柔暖的橘光，海面反射出断续的金橙高光，远处仍保留青绿色和深蓝色的冷静底色。

DdShell 是高频开发工具，不能为了氛围牺牲识别效率。所以这套主题没有把整页做成橙黄色蒙版，而是让橘色只出现在按钮、焦点、图表高亮、Logo 渐变和首页日落光源里。正文、面板、终端和列表仍以深海蓝、暖白和低饱和青色建立稳定对比。

## 视觉原则

1. 先保证可见度  
   正文使用深海蓝或暖白，按钮和图标有明确边界，橘色只做重点，不覆盖信息层。

2. 海面压住橘光  
   大面积背景由海青、深蓝和暖白承载，橘光作为日落方向的局部反射，避免界面泛黄。

3. 自然，但仍是工具  
   首页可以有海天线、日轮和水面反光；业务页面保持干净、低噪声、高密度，适合长期使用。

4. Logo 纹理来自海面  
   保留双 D 识别，但内部改为横向浪线、半落日和水面反光，不再复用其它主题的内部纹路。

## 色彩结构

- 日落：橘红、暖橙、浅金反光
- 海面：青绿、海蓝、深海蓝
- 表面层：暖白、沙白、浅米灰
- 强调色：日落橘 + 海面青
- 文本：深海蓝、灰蓝、暖白

## 适用场景

「橘子海」适合 DdShell 这类桌面开发工具的轻暖视觉方向：连接管理、终端、SFTP、监控、片段和工作流都可以共用。它希望在专业工具里提供更自然、更有记忆点的界面气质，同时保持清晰、耐看和可操作。
