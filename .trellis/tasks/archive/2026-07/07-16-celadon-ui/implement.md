# 实施计划：青瓷主题原型

1. 加载前端规范与 `ui/` 现有完整主题结构。
2. 以 Cloudrift 目录做机械结构复制，统一替换主题名称和品牌文案。
3. 重写双主题 tokens，建立青瓷配色、阴影和釉面变量。
4. 调整共享组件与布局细节，去除天空云层残留并加入瓷面质感。
5. 重写首页装饰和品牌说明，加入稀疏开片纹理。
6. 重绘浅色、暗色双版本 Logo。
7. 校验九个页面、文件引用、旧主题残留和格式。

## 验证命令

- `find ui/ui-celadon -type f | sort`
- 本地 `href` / `src` 引用检查
- `rg 'Cloudrift|云岚|cloud-high|cloud-low|sky-light' ui/ui-celadon`
- `git diff --check`

## 风险与回滚点

- 机械复制后必须完整清理旧主题名称与云层装饰，否则主题会只换色不换视觉语言。
- 裂纹只能作为低对比纹理，过密会影响工具页面可读性。
- 浅色文本与淡青表面需要保持足够对比，不能因追求瓷白而发灰。
