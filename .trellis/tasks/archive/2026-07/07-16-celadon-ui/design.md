# 技术设计：青瓷静态主题原型

## 结构来源

复制 `ui/ui-cloudrift/` 的完整页面与样式结构，保持九个 HTML 页面、四个共享 CSS、九个页面 CSS 和两个 Logo 的覆盖范围。复制仅作为结构起点，视觉令牌、首页装饰、品牌文案与 Logo 全部改为青瓷语言。

## 色彩系统

### 浅色

- 应用底：瓷白 `#EEF3EE`
- 基础面：暖瓷白 `#F7F8F3`
- 高层面：淡青釉 `#DDE8E1`
- 主交互：釉青 `#4F8579`
- 深交互：墨青 `#214C4A`
- 点睛：釉金 `#B79A62`
- 正文：深墨青 `#173B3A`

### 暗色

- 应用底：窑夜 `#081311`
- 基础面：深墨青 `#0D1D1A`
- 高层面：青黑釉 `#142824`
- 主交互：亮青釉 `#86B9A9`
- 点睛：低亮釉金 `#C2A66C`
- 正文：青瓷白 `#E8EFE9`

## 纹理策略

- 首页背景使用数条不规则 SVG/CSS 裂线与柔和釉面光斑，不铺满业务页面。
- 普通卡片只使用细内高光与双层边框表达瓷面厚度。
- Logo 内部的裂纹保持稀疏，并限制在线条内部，避免破坏双 D 识别。

## 差异化边界

- 不使用云岚的天空蓝、云带和漂浮雾层。
- 不使用墨纸的宣纸纤维、朱砂红和书写感。
- 不使用竹影的深绿竹节和竖向自然纹理。
- 不使用蓝栅的规则坐标网格与高亮电青线。

## 验证

- 对比参考目录文件集合。
- 检查所有本地 `href` / `src` 引用存在。
- 搜索旧主题名称、旧装饰类和旧核心调色残留。
- 检查 HTML 中 CSS 引用、主题切换和默认浅色状态。
